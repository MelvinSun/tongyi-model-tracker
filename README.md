## 🔧 技术栈

- Python 3.10+
- BeautifulSoup / Requests - 网页抓取
- GitHub Actions - 定时任务

## 📁 项目结构

```
.
├── scraper.py              # 主爬虫脚本
├── models.json             # 抓取的原始数据
├── .github/
│   └── workflows/
│       └── daily-scraper.yml  # 每日定时任务
├── requirements.txt        # Python 依赖
└── README.md              # 项目说明
```

## 🚀 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行爬虫
python scraper.py
```

## ⏰ 自动更新

项目配置了 GitHub Actions，每天北京时间早上 8:00 自动运行爬虫并更新 README。
